put a tweet about a certain word on your sidebar

== Description ==

Do you wanna put some twitter post that match a certain words? This might be the plugin for that.

== Installation ==

Just Download or install it from your wordpress dashboard. Activate, add the widget to your sidebar and remember to configure it.

OR

just add <?php twitterSay('ands keyword','ors keyword','number of tweets you want to display'); ?> to your template after activating the plugin. You can use space ti separate the keywords.
